Introduction

The Discord Bot that does it all. Automatically.

About this Bot

A fully customizable server moderation Discord bot for your Discord server that features a simple and intuitive web dashboard. Server management just got a whole lot easier.

Technology

The bot is bulit with python using the discord.py library, same with the API the bot uses.
The bot uses it's own API to filter and provide high quality images to your server. With the help of our super fast and flexible API we are able to provide you the best and fastest features.

Getting Started

To get started with our bot is fairly easy and we have done our best to make it simpler for you.

In-bot help

There is also a in-bot gettingstarted command that you can use if you know the installation process.
You also have the option to get help on all the bot commands right through discord with the use of the bot's internal help command.

Adding the bot

To add the bot to your server click here.

This does needs you to either be the owner of the server or have "manage server" permissions.

Commands

Syntax:
  • /
  • Moderation
    • Description : Let's you see moderation commands.
    Usage : /Moderation
    Note : The moderation commands only can be used by owner.
  • Giveaways
    • Description : Let's you Make Giveaways.
    Usage : /Giveaways
    Example : /giveaway start channel: prize: winners: duration:
    Note : The moderation commands only can be used by owner.
  • Reaction role
    • Description : Let's you make reaction role with button.
    Usage : /reactionrole-setup
    Example : /reactionroles-setup channel: role-1: role-2: role-3:
    Note : The moderation commands only can be used by owner.
  • Levels
    • Description : Let's you know your level and leaderboards.
    Usage : /level /levels
    Example : /level @user
  • Support
    • Description : Let's you make a ticket setup.
    Usage : /ticket
    Aliases : open and close ticket
